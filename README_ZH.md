# 如何使用
## 从[Release](https://github.com/zhaojh329/rttys/releases)页面下载编译好的程序或者自己编译

## 查看支持哪些命令行参数

	$ sudo ./rttys -h
	Usage of rttys:
      -conf string
            config file to load (default "./rttys.conf")
	  -cert string
	        certFile Path
	  -port int
	        http service port (default 5912)
      -ssl-cert string
            certFile Path
      -ssl-key string
            keyFile Path

## 运行

	sudo ./rttys
